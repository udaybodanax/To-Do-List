# ✅ To Do List

A highly functional task management application. This project handles CRUD operations entirely through JavaScript arrays and local persistent storage.

### ✨ Features
- Add, toggle completion, and delete tasks dynamically.
- Categorize tasks by priority levels (High, Medium, Low).
- Filter tasks by status (All, Ongoing, Completed).
- Real-time progress bar reflecting completion percentage.
- Search functionality to find specific tasks.

### 🛠️ Built With
- HTML5 & CSS3
- Vanilla JavaScript (LocalStorage API, Array Filtering/Mapping)

### 💻 How to Run
Just open the `index.html` file in your browser.
